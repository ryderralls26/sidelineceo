# FlagFooty - Quick Reference Guide\n\n## 🚀 Access Information\n\n**URL:** http://localhost:3000\n\n**Default Login:**\n- Email: `coach@example.com`\n- Password: `coach123`\n\n---\n\n## 📑 Navigation Structure\n\n| Tab | Purpose |\n|-----|---------|\n| **Home** | Landing page |\n| **Roster** | Manage players, positions, divisions |\n| **Schedule** | Games, awards, notes, invites |\n| **MGMT** | Positions, divisions, award types |\n| **Awards** | MVP leaderboard |\n| **Archive** | Finalized games |\n\n---\n\n## ⚠️ Key Features at a Glance\n\n### Schedule Page\n- ✅ Time displays next to Location (grey text)\n- ✅ Game Notes button (coach only)\n- ✅ MVP awards button (coach only)\n- ✅ Award winners shown below finalized games\n- ✅ Invite section at bottom (coach only)\n\n### Roster Page\n- ✅ Player table with columns:\n  - Play, 4Q Lock, Jersey #\n  - First Name, Last Name, Nickname\n  - Offensive Position, Defensive Position\n  - **Secondary Position** (new)\n  - **Division** (new)\n  - Actions\n- ✅ Buttons: Add Player, Generate Lineup, Save, Finalize\n- ✅ Coach selection (Coach 1 or Coach 2)\n- ✅ Live game card previews\n\n### MGMT Page\n- ✅ **Positions** table (name, abbreviation, rank)\n- ✅ **Divisions** table (name, abbreviation)\n  - Default: KIND, FR, SOPH, JR, SR\n- ✅ **Awards** management (MVP, POYG, etc.)\n\n### Awards Page\n- ✅ **Leaderboard** matrix view\n- ✅ Player names (rows) × Award types (columns)\n- ✅ Award count tallies\n- ✅ Sorted by total awards (descending)\n- ✅ Gold/Silver/Bronze medals for top 3\n\n---\n\n## 📋 Common Tasks\n\n### Add a New Player\n1. Go to **Roster** tab\n2. Click **\"Add Player\"**\n3. Fill in details (jersey, name, positions, division)\n4. Click **\"Save Changes\"**\n\n### Schedule a Game\n1. Go to **Schedule** tab\n2. Click **\"Add Game\"**\n3. Enter date, opponent, location, field, time\n4. Click **\"Add Game\"**\n\n### Award MVP\n1. Go to **Schedule** tab\n2. Find the game\n3. Click **\"MVP\"** button\n4. Select player and award type\n5. Enter reason (optional)\n6. Click **\"Award\"**\n\n### Add Game Notes\n1. Go to **Schedule** tab\n2. Click **\"Game Notes\"** button on any game\n3. Type coach notes\n4. Click **\"Save Notes\"**\n\n### Configure Divisions\n1. Go to **MGMT** tab\n2. Scroll to **Division Management**\n3. Click **\"Add Division\"**\n4. Enter name and abbreviation\n5. Click **\"Add Division\"**\n\n### Configure Award Types\n1. Go to **MGMT** tab\n2. Scroll to **Awards Management**\n3. Click **\"Add Award Type\"**\n4. Enter award name and description\n5. Click **\"Add Award\"**\n\n### Send Team Invite\n1. Go to **Schedule** tab\n2. Scroll to bottom (**Invite Team Members**)\n3. Click **\"Send Invite\"**\n4. Enter email, select role (Parent/Player)\n5. Optional: Grant admin access (for parents)\n6. Click **\"Send Invite\"**\n\n### Finalize a Game\n1. Go to **Roster** tab with gameId parameter\n2. Generate lineup\n3. Save roster\n4. Award MVPs from Schedule page\n5. Click **\"Finalize\"** button in Roster\n6. Game moves to Archive\n\n### View Leaderboard\n1. Go to **Awards** tab\n2. View player rankings\n3. See award breakdown by type\n\n---\n\n## 🎨 Visual Indicators\n\n### Status Badges\n- 🟡 **Yellow:** Pending (invites, scheduled)\n- 🟢 **Green:** Completed/Accepted\n- 🔵 **Blue:** Finalized\n- 🔴 **Red:** Error/Delete actions\n\n### Medals (Leaderboard)\n- 🥇 **Gold:** 1st place\n- 🥈 **Silver:** 2nd place\n- 🥉 **Bronze:** 3rd place\n- **Numbers:** 4th place and below\n\n### Button Colors\n- **Green Gradient:** Primary actions (Add, Save, Award)\n- **Blue:** Secondary actions (Finalize, Generate)\n- **Grey:** Neutral actions (Cancel, View)\n- **Red:** Destructive actions (Delete)\n\n---\n\n## 📊 Data Structure\n\n### Player Fields\n```\n- ID\n- Play (checkbox)\n- 4Q Lock (checkbox)\n- Jersey Number\n- First Name\n- Last Name\n- Nickname\n- Offensive Position\n- Defensive Position\n- Secondary Position (new)\n- Division (new)\n```\n\n### Game Fields\n```\n- Date\n- Opponent\n- Location\n- Field\n- Time\n- Status (scheduled/completed)\n- Result (for completed games)\n- Notes (coach only)\n```\n\n### Division Fields\n```\n- Name (e.g., \"Freshman\")\n- Abbreviation (e.g., \"FR\")\n```\n\n### Award Fields\n```\n- Player ID\n- Award Type (e.g., \"MVP\")\n- Game ID\n- Notes/Reason\n- Awarded By (coach)\n- Awarded At (timestamp)\n```\n\n---\n\n## 🔒 Permissions\n\n### Coach (Default User)\n- ✅ Full access to all features\n- ✅ Add/Edit/Delete players, games, divisions\n- ✅ Award MVPs\n- ✅ Send invites\n- ✅ View/edit all data\n\n### Parent (Admin)\n- ✅ View roster and schedule\n- ✅ Edit team data\n- ❌ Cannot delete critical data\n\n### Parent (Non-Admin)\n- ✅ View roster and schedule\n- ❌ Cannot edit\n\n### Player\n- ✅ View their own stats\n- ❌ Cannot edit\n\n---\n\n## 💾 Data Storage\n\n**Location:** Browser LocalStorage\n\n**Keys:**\n- `roster` - Player data\n- `games_schedule` - Game schedule\n- `positions` - Position definitions\n- `divisions` - Division definitions\n- `award_types` - Award type definitions\n- `player_awards` - Award records\n- `team_coaches` - Coach names and selection\n- `team_invites` - Invite records\n- `finalized_games` - Archived games\n- `game_notes` - Per-game coach notes\n\n**Backup:** Export via browser DevTools → Application → LocalStorage\n\n---\n\n## 🐞 Troubleshooting\n\n### Server Not Starting\n```bash\n# Kill any process on port 3000\nkillall -9 node\n# Restart server\nnpm run dev\n```\n\n### Data Not Saving\n- Check browser console for errors\n- Verify LocalStorage is enabled\n- Clear browser cache\n- Try incognito/private mode\n\n### Changes Not Appearing\n- Hard refresh: Ctrl+Shift+R (Windows/Linux) or Cmd+Shift+R (Mac)\n- Clear cache\n- Restart dev server\n\n### TypeScript Errors\n```bash\n# Rebuild project\nnpm run build\n# Check for type errors\nnpx tsc --noEmit\n```\n\n---\n\n## 📱 Mobile Support\n\nThe application is **fully responsive** and works on:\n- ✅ Desktop browsers\n- ✅ Tablets (landscape recommended)\n- ✅ Mobile phones (portrait/landscape)\n\n**Best Experience:**\n- Desktop: Chrome, Firefox, Edge\n- Mobile: Safari (iOS), Chrome (Android)\n\n---\n\n## ⌨️ Keyboard Shortcuts\n\n| Key | Action |\n|-----|--------|\n| `Esc` | Close modal |\n| `Tab` | Navigate form fields |\n| `Enter` | Submit form |\n| `Space` | Toggle checkbox |\n\n---\n\n## 🎯 Workflows\n\n### Game Day Workflow\n1. **Before Game:**\n   - Review roster on Roster page\n   - Generate lineup\n   - Print game cards\n\n2. **During Game:**\n   - Use printed coach/ref cards\n   - Track player performance\n\n3. **After Game:**\n   - Award MVPs from Schedule page\n   - Add game notes\n   - Finalize game\n\n### Season Management Workflow\n1. **Setup:**\n   - Configure divisions (MGMT page)\n   - Configure award types (MGMT page)\n   - Add players (Roster page)\n\n2. **Regular Season:**\n   - Add games (Schedule page)\n   - Run game day workflow\n   - Monitor leaderboard (Awards page)\n\n3. **End of Season:**\n   - Review final leaderboard\n   - Export data backup\n   - Archive season\n\n---\n\n## 💡 Tips & Best Practices\n\n### Player Management\n- ✅ Assign all three positions for flexibility\n- ✅ Set divisions for accurate card printing\n- ✅ Use descriptive nicknames\n- ✅ Keep jersey numbers unique\n\n### Game Management\n- ✅ Add games early for planning\n- ✅ Use game notes for strategy\n- ✅ Award MVPs immediately after games\n- ✅ Finalize games once stats are confirmed\n\n### Award Management\n- ✅ Create award types before season starts\n- ✅ Be consistent with award names\n- ✅ Add reasons for player development feedback\n- ✅ Review leaderboard regularly\n\n### Division Management\n- ✅ Use standard abbreviations (KIND, FR, SOPH, JR, SR)\n- ✅ Keep division names clear\n- ✅ Don't delete divisions mid-season\n\n---\n\n## 📞 Support\n\n### Documentation\n- `flagfooty_IMPLEMENTATION_SUMMARY.md` -
